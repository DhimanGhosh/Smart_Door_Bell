# Smart_Door_Bell

## Components
#### Required
- Raspberry Pi 4B (any version will support)
- Micro SD Card
- HC-SR04
- PIR Motion Sensor
- Piezzo Buzzer
- Jumper Cables
#### Optional
- LED
- 47 Ohm Registor
- Breadboard
- WiFi / Ethernet
- Mouse
- Keyboard
- Monitor
- Micro HDMI to HDMI cable

## Execute
> Note: Edit [config.ini](https://github.com/DhimanGhosh/Smart_Door_Bell/config.ini) and enter the GPIO pins of your setup
```sh
chmod +x run.sh
./run.sh
```
